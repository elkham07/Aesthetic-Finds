## 📖 About the Project

**Aesthetic Finds** is a minimalist platform designed for curating and showcasing premium home decor. It allows users to build a personalized gallery of products integrated with **Amazon**, turning a simple product list into a magazine-worthy sanctuary.

The goal is to provide a clean, "premium" UI where every added piece transforms the space.

## 🚀 Key Features

* **Product Curation:** Seamlessly add and manage multiple products via the **Admin View**.
* **Amazon Integration:** Each item includes a direct link to Amazon, making it easy for visitors to purchase.
* **Dynamic Gallery:** Real-time updates to the product collection visible to all users.
* **Minimalist UI:** High-impact design focused on white space and elegant typography.
